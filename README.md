This project is a sophisticated, single-page interactive report designed to visualize the progress and future potential of six core sectors critical to Bihar's economic development from 2010 to 2030.

The dashboard transforms complex economic data into an intuitive, interactive experience, featuring AI-powered tools for policy and investment planning.

‚ú® Key Features

Six Core Sectors: Detailed analysis of FMCG & Agriculture, Healthcare, Education, IT & ITES, Logistics & Supply Chain, and Renewable Energy (RE).

Dynamic Comparison Tool: Allows users to select any two sectors and instantly compare their 2025 status and 2030 growth projections on a dynamic chart.

Gemini AI Integration (LLM/Search Grounding): Generates formal policy brief drafts for each sector's key recommendation, contextualized by real-time data access.

Text-to-Speech (TTS) Pitch Generator: Creates a concise, professional, 30-second audio investment pitch for the current sector using the Gemini TTS API.

Responsive UI: Built as a single-page application (SPA) using a clean, modern design for optimal viewing on desktop, tablet, and mobile.
